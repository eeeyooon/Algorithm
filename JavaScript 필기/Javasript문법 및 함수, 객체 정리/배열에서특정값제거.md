## 배열에서 특정값 제거하기

### 맨 앞, 맨 뒤, 중간 값 제거하기

<br/>

    배열의 맨 앞에 값 추가 : unshift()
    배열의 맨 앞 값 제거 : shift()
    배열의 맨 뒤에 값 추가 : push()
    배열의 맨 뒤 값 제거 : pop()
    배열의 중간에 값 추가, 삭제 : splice()


<br/>

#### `unshift()`

- 배열의 맨 앞 파라미터로 전달 받은 element들을 추가한다.
- 파라미터로 전달된 elements가 추가 된, 새로운 배열의 길이를 리턴한다.

<br/>

#### `shift()`

- 배열의 가장 첫 element를 삭제한다.
- 삭제한 element를 리턴한다.

<br/>

#### `push()`

- 배열의 맨 끝에 새로운 element를 추가한다.
- 파라미터로 전달된 elements가 추가 된, 새로운 배열의 길이를 리턴한다.

<br/>

#### `pop()`

- 배열의 가장 뒤에 있는 element를 삭제한다.
- 삭제한 element를 리턴한다.

<br/>
